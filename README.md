# Operation Tracking Dashboard React

This is a React conversion of the uploaded group project.

## Pages
- `/` home links page
- `/login` login page
- `/welcome` welcome page
- `/dashboard` incident dashboard with JSON loading and filters
- `/tickets` ticket management page with generated tickets and edit modal

## Run
```bash
npm install
npm run dev
```

## Notes
- `public/tickets.json` is used by the dashboard route.
- `public/assets/login-welcome/Images` contains the copied original images.
- The dashboard and ticket management logic were converted from vanilla JavaScript to React state.
